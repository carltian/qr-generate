## 字号调整计划（基准16px）

在Taro 375设计稿下，16px = 32rpx

### 字号规范
| 用途 | px | rpx |
|------|-----|------|
| 小字/提示 | 12px | 24rpx |
| 正文 | 14px | 28rpx |
| 大正文 | 16px | 32rpx |
| 小标题 | 18px | 36rpx |
| 标题 | 20px | 40rpx |
| 大标题 | 24px | 48rpx |

### 调整内容

#### 全局样式 (app.scss)
- 基础字体：24px → 28rpx (14px)

#### 生成页面 (generate/index.scss)
- 标题：36px → 48rpx (24px)
- 副标题：24px → 28rpx (14px)
- 标签：24px → 28rpx (14px)
- 输入框：26px → 28rpx (14px)
- 提示文字：22px → 24rpx (12px)
- 按钮：26px → 32rpx (16px)
- 预览标题：24px → 28rpx (14px)
- 占位文字：24px → 28rpx (14px)

#### 配置页面 (configure/index.scss)
- 卡片标题：24px → 32rpx (16px)
- 表单标签：22px → 28rpx (14px)
- 输入框：24px → 28rpx (14px)
- 单选项：22px → 28rpx (14px)
- 标签文字：22px → 24rpx (12px)
- 保存按钮：26px → 32rpx (16px)
