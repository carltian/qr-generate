## UI调整计划

### 问题分析
当前UI元素尺寸过大，导致文字和按钮溢出屏幕

### 调整方案

#### 生成页面 (generate/index.scss)
- 标题字体：48px → 36px
- 副标题：28px → 24px
- 输入区域padding：32px → 24px
- 输入框最小高度：200px → 120px
- 按钮高度：96px → 80px
- 按钮字体：32px → 28px
- 预览区域padding：32px → 24px
- 二维码预览尺寸：280px → 200px

#### 配置页面 (configure/index.scss)
- 卡片padding：28px → 20px
- 卡片标题：30px → 26px
- 表单标签：26px → 24px
- 输入框高度：80px → 64px
- 按钮高度：96px → 72px
- Logo预览区域：120px → 80px
- 颜色预览：80px → 60px
- 保存按钮区域padding：24px → 16px

#### 全局样式 (app.scss)
- 基础字体：28px → 26px
